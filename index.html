OneDrive Client for Linux

A complete tool to interact with OneDrive on Linux

 
Build and Install from Github Run via Docker

OneDrive Client Features

The OneDrive client provides the following features and capabilities

Platform Support

Supports most Linux platforms and others such as FreeBSD across multiple architectures (i686, x86_64, ARMHF & ARM64)

OneDrive Account Support

Works with OneDrive Personal, OneDrive for Business, Office365 and SharePoint Shared Libraries

Data Integrity

Designed to ensure data integrity for your data - local state caching, file upload / download validation, resumable uploads

Test before Run

Test your changes before actually running the configuration, Prevent major accidental data deletion after client configuration change
